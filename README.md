# mobile-login-signup-ui
A clean mobile UI design featuring a Login screen and Create Account (Sign Up) screen with Indian phone number input, social login icons, and a purple-pink theme.
demo video https://youtu.be/ySDmKBT4qa4 This project contains a two-screen mobile authentication UI design — Login and Create Account — built as a Figma mockup. The design includes:

Login screen

IND +91 phone number input field
Password field with lock icon
Forgot password link (purple)
Login button and Sign up redirect
Social login — Facebook, Google, Instagram
Create Account screen

Name, Email/Phone, and Password input fields
Sign Up button and Sign in redirect
Back navigation arrow
Social login — Facebook, Google, Instagram
Design highlights

Purple and pink color theme
Soft pink circular blob background element
Clean white minimal layout
Mobile screen with status bar (10:25 / 10:40)
